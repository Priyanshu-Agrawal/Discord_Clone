## Running this webApp

To run the demo follow these steps:

1.Make sure node is installed in your system

```sh
    npm install
    npm run start
```

## Technology

This Project uses:

- CometChat Pro 3.0.0
- CometChat UI Kit
- Firebase
- React.js
- Uuid
- Validator
- @emotion/core
- dateformat
